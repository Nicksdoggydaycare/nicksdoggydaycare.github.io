# Nick's Doggy Daycare — Project Meadow

A custom static website for Nick's Doggy Daycare.

## Files

- `index.html` — page structure and website copy
- `styles.css` — colors, layout, typography, and responsive design
- `script.js` — navigation, scroll animations, and live weather
- `assets/images/` — replace placeholder image files here
- `assets/video/hero.mp4` — add the finished hero video here

## Before launch

1. Replace the placeholder email address.
2. Add the business phone number and Instagram.
3. Add original property, dog, transportation, and boarding photos.
4. Add `assets/video/hero.mp4`.
5. Add a hero poster image named `assets/images/hero-poster.jpg`.
6. Review all service details and policies.

## GitHub Pages

Upload all files to the repository root. GitHub Pages will use `index.html` as the homepage.
